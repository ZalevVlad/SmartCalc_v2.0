SmartCalc

Для переключения между режимами калькулятора используйте меню.

Калькулятор поддерживает следующие операции:

  • Арифметические операторы:

    Название оператора Выражение
    Скобки (a + b)
    Сложение a + b
    Вычитание a - b
    Умножение a * b
    Деление a / b
    Возведение в степень a ^ b
    Остаток от деления a % b
    Унарный плюс + a
    Унарный минус - a

  • Функции:

    Описание функции Функция
    Вычисляет косинус cos(x)
    Вычисляет синус sin(x)
    Вычисляет тангенс tan(x)
    Вычисляет арккосинус acos(x)
    Вычисляет арксинус asin(x)
    Вычисляет арктангенс atan(x)
    Вычисляет квадратный корень sqrt(x)
    Вычисляет натуральный логарифм ln(x)
    Вычисляет десятичный логарифм log(x)

    Для вычисления в режиме Calculator используйте кнопку =.

    Для построения графика в режиме Graph используйте кнопку plot.
